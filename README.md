# PackScript

A simple tool to organize ItemsAdder resource packs by moving models and textures into separate folders.

## Quick Start

1. Download `main.py`
2. Run with [uv](https://github.com/astral-sh/uv) (recommended):
```bash
uv run main.py
```

Or with regular Python:
```bash
python3 main.py
```
